# LingoFlow 视觉重设计 - 「流动的语言」

## 🎨 设计理念

**核心概念**：翻译是语言的流动，界面应该体现这种流畅性和优雅感。

### 美学方向

- **字体系统**：
  - 主字体：**DM Sans** - 现代、优雅、可读性强
  - 等宽字体：**JetBrains Mono** - 用于快捷键和代码显示
  
- **色彩哲学**：
  - 保持浅色主题，但引入**微妙的渐变**
  - **玻璃态质感** (Glassmorphism) - 半透明背景 + 模糊效果
  - **柔和的色彩过渡** - 避免生硬的颜色跳跃

- **动效设计**：
  - **流体动画** - 使用 cubic-bezier 缓动函数
  - **错开动画** (Staggered Animation) - 卡片依次出现
  - **弹性过渡** (Bounce Transition) - 开关和按钮的回弹效果
  - **脉冲动画** (Pulse Animation) - 录制状态的呼吸效果

- **空间布局**：
  - **充足的留白** - 元素之间有呼吸感
  - **层次分明** - 通过阴影和透明度建立视觉层级
  - **微妙的背景纹理** - 径向渐变营造氛围

## ✨ 关键视觉特性

### 1. 设置界面

#### 标签页导航
- **流体胶囊设计**：圆角胶囊形状，激活时有渐变蓝色背景
- **玻璃态背景**：半透明白色 + 背景模糊 (backdrop-filter)
- **悬浮光晕**：hover 时出现径向渐变光效
- **图标动画**：hover 时图标放大 (scale 1.1)
- **激活状态**：渐变背景 + 内阴影 + 外发光

#### 快捷键设置
- **玻璃态卡片**：半透明渐变背景 + 模糊效果
- **悬浮光效**：hover 时从左到右的光线扫过
- **录制状态**：脉冲动画 + 蓝色光晕扩散
- **等宽字体**：快捷键显示使用 JetBrains Mono

#### 开关组件
- **流体切换**：弹性过渡动画 (cubic-bezier bounce)
- **渐变背景**：开启时蓝色渐变 + 外发光
- **立体阴影**：内阴影 + 外阴影营造深度

### 2. 翻译界面

#### 源文本卡片
- **玻璃态设计**：半透明白色 + 12px 背景模糊
- **悬浮光效**：hover 时出现径向渐变光晕
- **操作按钮**：渐变背景 + hover 时变为实心蓝色

#### 语言切换栏
- **流体胶囊**：圆角胶囊形状 + 玻璃态背景
- **互换按钮**：圆形按钮 + 180° 旋转动画 + 放大效果
- **hover 状态**：渐变蓝色背景 + 白色文字

#### 翻译结果卡片
- **品牌色左边框**：3px 左边框显示服务商颜色
- **品牌色光晕**：hover 时左侧出现模糊的品牌色光效
- **错开动画**：卡片依次出现 (50ms 间隔)
- **悬浮效果**：hover 时上移 3px + 右移 2px
- **玻璃态背景**：半透明白色 + 背景模糊

#### 加载状态
- **脉冲动画**：卡片整体透明度呼吸效果
- **旋转器**：圆形边框旋转动画

#### 错误状态
- **红色主题**：浅红色渐变背景 + 红色左边框
- **柔和阴影**：红色调阴影

## 🎯 技术实现亮点

### CSS 技术
1. **backdrop-filter: blur()** - 玻璃态模糊效果
2. **linear-gradient()** - 微妙的渐变背景
3. **radial-gradient()** - 光晕和氛围效果
4. **cubic-bezier()** - 自定义缓动函数
5. **animation-delay** - 错开动画时序
6. **box-shadow (multiple)** - 多层阴影营造深度
7. **inset shadow** - 内阴影增强立体感
8. **::before pseudo-element** - 装饰性光效层

### 动画曲线
- **Fast**: `cubic-bezier(0.4, 0, 0.2, 1)` - 快速响应
- **Normal**: `cubic-bezier(0.4, 0, 0.2, 1)` - 标准过渡
- **Bounce**: `cubic-bezier(0.34, 1.56, 0.64, 1)` - 弹性效果
- **Smooth**: `cubic-bezier(0.16, 1, 0.3, 1)` - 流畅缓入缓出

## 🌟 用户体验提升

1. **视觉反馈**：每个交互都有明确的视觉响应
2. **流畅动画**：所有过渡都使用精心调校的缓动函数
3. **层次分明**：通过阴影、透明度、模糊建立清晰的视觉层级
4. **品牌识别**：服务商颜色让用户快速识别翻译来源
5. **呼吸感**：充足的留白和柔和的背景让界面不拥挤
6. **细节精致**：内阴影、外发光、渐变等细节提升质感
